# levi-chatbot

A small Node.js chatbot server for serving portfolio knowledge.

## Overview

This repository contains a simple chatbot server that uses local knowledge (see `portfolioKnowledge.js`) and an entry point at `index.js`.

## Files

- `index.js`: Main server/entry script.
- `package.json`: Project metadata and dependencies.
- `portfolioKnowledge.js`: Local knowledge/data used by the chatbot.

## Requirements

- Node.js (v14+ recommended)

## Install

Run:

```
npm install
```

## Run

Start the server with:

```
node index.js
```
